Nápověda

Klikněte na ikonu měsíce/slunce v pravém horním rohu navigace. Dark mode se automaticky uloží do vašeho prohlížeče.

Ano, web je plně responzivní a přizpůsobuje se různým velikostem obrazovek od mobilních zařízení po desktop.

Použijte kontaktní formulář na stránce Kontakt. Vyplňte požadované údaje a odešlete zprávu.

Projekt využívá Vite pro build, Tailwind CSS v4 pro styling, Alpine.js pro interaktivitu a PHP pro zpracování formulářů.
